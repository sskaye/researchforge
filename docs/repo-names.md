# Candidate Repository Names

## On LitForge — too crowded to recommend

LitForge looked attractive, but a search reveals enough active collisions that I would not pick it. The disqualifying one is **litforge.io**, which is an active, marketed *AI-powered* storytelling platform — same broad category (AI tools for literature/text) and same one-word domain. Anyone Googling our project would land on theirs first. Beyond that:

- **LitForge Press** — a publishing imprint with 35 books on Goodreads, Audible, Patreon, and Facebook presence.
- **LitForge-AI-epub2pdf** — a GitHub repo, also AI-adjacent.
- **LeSoviet/litforge** — another GitHub document-library project.
- **Litforge** — Amazon brand for plumbing fittings (semantically distant, but it shows the bare word is in commercial use).

Different category collisions you can usually shrug off; the litforge.io collision you cannot.

## XForge alternatives — vetted

I checked each of these for active GitHub projects, products, and brand uses. Status notes are based on a quick search and should be re-verified at the moment of registration.

### Shortlist (clean and recommended)

- **CorpusForge** — Recommended. *Corpus* is the right word — it's the technical term scientists use for "the body of literature you're working from," and forging a corpus into something useful is exactly what the suite does. No active GitHub or product collisions found. Distinctive, memorable, reads well as `github.com/<you>/corpusforge`.

- **EvidenceForge** — Strong second. Captures the engineering register the user is going for ("evidence, risk, what would need to be developed") better than corpus-anything. No collisions found. Slightly heavier than CorpusForge but more on-message.

- **TomeForge** — Distinctive and evocative. A *tome* is a substantive scholarly volume — aligns well with "report, not answer" framing. No collisions found. Slight whiff of fantasy-genre, which is fine in moderation.

- **ScholarForge** — Clean and clear. Skews more academic than engineering. No collisions found.

- **InquiryForge** — Socratic feel. "Inquiry" maps to "specify a question precisely," which is the user's framing exactly. No collisions found. A little long.

### Avoid (collisions)

- **PaperForge** — Heavily taken. There is a GitHub project at `QJHWC/PaperForge` that is itself an AI-powered academic paper writing system (idea generation, literature search, experiments, LaTeX compilation) — directly competing semantic territory. Also `paperforge.app` (Notion converter), `papersforge.com` (paper-writing service), and a gaming content brand. Disqualified.
- **SciForge** — Taken by a [GFZ Potsdam project](https://www.gfz-potsdam.de/en/centre/escience-centre/projects/sciforge/) on scientific software publishing — and the project is explicitly named after SourceForge for adjacent science/research-tools reasons. Awkward overlap.
- **SynthForge** — Taken by a 2024 facial-dataset synthesis project (GitHub org `synth-forge`, project page at synth-forge.github.io).
- **OpusForge** — Taken by a developer portfolio generator (`opusforge.tech`, GitHub `kbvkishore/OpusForge`).
- **SourceForge** — Famously taken (the original software forge).

### Other XForge ideas that *might* be clean (would need verification)

These appeared not to surface in quick searches, but I would do a deeper check before committing:

- **AssayForge** — *Assay* is a scientifically loaded word (a test, an analysis); the connection to "assess" is intuitive.
- **DossierForge** — A *dossier* is a curated collection of documents on a topic; very on-the-nose for what skill 1 produces.
- **TreatiseForge** — Captures the long-form serious-report end of the spectrum.
- **PrimerForge** — A *primer* is exactly the kind of mid-length technical brief the suite produces.
- **DigestForge** — A *digest* of literature; clean and short.

## Recommendation

If you want the safest bet that still reads well: **CorpusForge**.
If you want the name to lean toward your engineering/decision-support framing: **EvidenceForge**.
If you want something distinctive with a slight literary character: **TomeForge**.

All three are vetted clean as of this search and would pair naturally with skill names like `corpusforge-search`, `corpusforge-summarize`, `corpusforge-extract`, `corpusforge-report`.
